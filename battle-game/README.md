# 🎮 Battle Royale — 多人联机大逃杀

Node.js + WebSocket + HTML5 Canvas 实时多人对战游戏

## 快速启动

```bash
cd battle-game
npm install
./start.sh          # 或 node server.js
```

浏览器打开 `http://localhost:3000`

## 多人测试（AI机器人）

```bash
# 先启动服务器（另一个终端）
./start.sh

# 启动5个AI机器人自动对战
node test_bots.js
```

机器人在同一局游戏中自动移动、射击、投雷，直至决出胜者。

## 操作说明

| 按键 | 功能 |
|------|------|
| W/A/S/D 或 方向键 | 移动 |
| 鼠标移动 | 瞄准 |
| 鼠标按住 | 射击（自动连发） |
| **G** | 投掷手雷 |
| 走到物品上 | 自动拾取 |
| 死亡后 | 自动进入观战模式 |

## 物品

| 物品 | 效果 |
|------|------|
| 血包 (+) | 回复 25 HP（医疗兵 40） |
| 护甲 (A) | 获得 25 护甲（减伤 50%） |
| 手雷 (G) | 获得 1 枚手雷 |
| 弹药 | 补充当前武器弹药 |

## 手雷

- 按 G 键向鼠标方向投掷
- 80 像素爆炸半径
- 中心伤害 50，边缘递减
- 受护甲减伤影响

## 游戏机制

### 缩圈
- 起始半径 1800，持续缩小至 120
- 圈外每秒扣 1.5 HP
- 距离圈边 <80px 时屏幕红色闪烁警告

### 掩体
- 地图有 17 栋建筑
- 子弹无法穿透建筑
- 小地图显示建筑位置（灰色方块）

### 结算
- 获胜：200 + 击杀数×30 金币
- 可用于商店购买新角色

## 项目结构

```
battle-game/
├── server.js          # 后端（Express + WebSocket）
├── package.json       # 依赖
├── start.sh           # 启动脚本
├── test_bots.js       # AI机器人测试
├── README.md          # 本文档
└── public/
    ├── index.html     # 前端 SPA
    ├── css/game.css   # 样式
    └── js/            # JS 已内联在 HTML
```

## 技术栈

- **后端**: Node.js + Express + ws (WebSocket)
- **前端**: HTML5 Canvas + 原生 JavaScript
- **数据**: 内存存储（重启清空）
- **同步**: 60fps WebSocket 广播
- **通信**: 仅发送玩家视野内数据
